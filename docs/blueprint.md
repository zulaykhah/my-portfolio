# **App Name**: Wuraola's Portfolio

## Core Features:

- Profile Section: Display a professional headshot and a brief introduction highlighting key skills and experience, pulled from the resume.
- Projects Showcase: Visually showcase projects completed, linking to GitHub repositories or live demos where applicable.
- Skills Overview: Display technical skills, certifications, and proficiencies using visual charts or progress bars.
- Resume Integration: Automatically extract key information (experience, education, skills) from a provided resume document (resume.pdf) and display it in designated sections.
- Contact Form: Allow visitors to send messages directly, with form validation and spam protection.
- LinkedIn Integration: Make prominent badge/link to LinkedIn profile, displayed throughout the whole application

## Style Guidelines:

- Primary color: Deep blue (#313552) to evoke trust and professionalism.
- Background color: Light gray (#E5E6E6), provides a clean and modern backdrop.
- Accent color: Teal (#2E8A99) to highlight important links, buttons, and interactive elements, without defaulting to the user's example.
- Body font: 'Inter', sans-serif, for readability and a clean, modern aesthetic.
- Headline font: 'Space Grotesk', sans-serif, providing a modern and tech-oriented feel, contrasting smoothly with the body.
- Use minimalist line icons to represent skills and technologies.
- Employ a single-page layout with clear sections, smooth scrolling, and a responsive design.
- Incorporate subtle animations, such as fade-in effects or progress bar updates, to enhance user experience.
